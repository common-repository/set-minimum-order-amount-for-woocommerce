Set Minimum Order Amount for your Woocommerce store, you can set any custom amount you want!

== Description ==
This plugin will allow you to set a custom order amount, going from 0 to a billion euro's and more!
Doing this is very usefull to avoid small orders that give more overhead then profit.

Simply install, and activate this lightweigth plugin, go to the general Woocommerce settings page and enter your Minimum Order Amount.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Woocommerce settings page, in the General menu add your Minimum Order Amount.

== Frequently Asked Questions ==
= How to set a minimum order amount? =
It is easy!
Go to the Woocommerce settings page, in the General menu, add your Minimum Order Amount.
= What happens if I do no set a value? =
We programmed it that the default value is 0, so it will allow all orders until a custom value is set.
= What happens if I uninstall? =
No longer needed? Then deactivate and uninstall the plugin, we will remove the data from Wordpress. So next time you will need to set a new custom amount.
= Can you translate? =
Sure, if you provide the translation to us, we will add it to the plugin core.
